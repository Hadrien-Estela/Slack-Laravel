    • $external_id — Slack\Objects\string|null
      A custom identifier that must be unique for all views on a per-team basis.
    • $notify_on_close — Slack\Objects\boolean|null

    • $private_metadata — Slack\Objects\string|null

    • $submit — \Slack\Objects\CompositionObjects\Text|null
      An optional plain_text element that defines the text displayed in the submit button at the bottom-right of the view. submit is required when an input block is within the blocks array. Max length of 24 characters.
    • $title — \Slack\Objects\CompositionObjects\Text
